## Описание

Random Password Generator — это графическое приложение на Python для генерации безопасных случайных паролей с возможностью настройки параметров и сохранения истории генерации.

### Основные возможности

- **Настройка длины пароля** (от 4 до 50 символов) с помощью ползунка
- **Выбор типов символов**:
  - Цифры (0-9)
  - Буквы (A-Z, a-z)
  - Специальные символы (!@#$%^&*()_+=-[]{};:?/.>,<)
- **Генерация случайных паролей** с использованием библиотеки `random`
- **Копирование пароля** в буфер обмена одним кликом
- **Автоматическое сохранение истории** в JSON файл
- **Просмотр истории** всех сгенерированных паролей в таблице
- **Очистка истории** при необходимости

## Требования

- Python 3.6 или выше
- Стандартные библиотеки: `tkinter`, `random`, `string`, `json`, `os`, `datetime`
